# MathGame

Beginner-friendly math learning app built with **HTML, CSS, and vanilla JavaScript**.

## Run

- Open `index.html` in your browser.

## Deploy on Render

### Option A (recommended): Render “Static Site”

- In Render dashboard, click **New** → **Static Site**
- Connect your GitHub repo: `halyapabba2014-creator/MathGame`
- **Branch**: `main`
- **Build Command**: *(leave empty)*
- **Publish Directory**: `.`
- Click **Create Static Site**

### Option B: Blueprint (uses `render.yaml`)

- In Render dashboard, click **New** → **Blueprint**
- Select this repo and approve
- Render will read `render.yaml` and deploy automatically
